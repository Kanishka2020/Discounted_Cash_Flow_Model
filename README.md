# Discounted_Cash_Flow_Model

#### The Discounted Cash Flow (DCF) model  is one of the most widely used methods for valuing a company. It’s based on a simple idea:

### A company is worth the present value of all the cash it will generate in the future.

## What is the DCF Model?

DCF estimates a company’s intrinsic value by:

1. Forecasting future free cash flows (FCF)
2. Discounting them back to today using a required rate of return (discount rate)

It relies on the Time Value of Money, which says money today is worth more than money in the future.

## Types of DCF

There are two main versions:

1. FCFF (Free Cash Flow to Firm) → values entire business (debt + equity)
2. FCFE (Free Cash Flow to Equity) → values only equity shareholders

Most common in practice: FCFF model
